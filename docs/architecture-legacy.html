L1

Physics Engine

The "Green Audit" — verifies kWh dispensed vs. kWh received via telematics data.

Variance <15% Telematics

Physics Verification Formula

Variance = |Edispensed - (ΔSoC × BatteryCapacity)| / Edispensed
If Variance < 15%, transaction is VERIFIED
