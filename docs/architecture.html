10-Layer Microservices Architecture

Token Bridge

Commerce Engine

Stripe OCPI CDR
L8

Energy Manager

Modbus BACnet

Device Gateway

Hardware abstraction layer for OCPP, OCPI roaming, and ISO 15118 Plug&Charge.

Engagement
